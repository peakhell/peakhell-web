写给快要毕业的自己

​ 不知不觉又到了毕业季, 不同于以往送别师兄师姐, 这一次,真的是自己毕业了.

​ 实习, 做毕设, 答辩, 就业派遣. 从3月份开始, 便开始忙碌着这些事情, 仿佛一切都是那么自然. 也是,我也不过是跟其他所有的毕业生一样, 忙碌着学校为我们做的安排罢了. 只是偶尔在学校开始忙完碌毕设等事情回到公司时, 总是有点不太想走, 而只要再过一个月, 这座我生活了4年的宿舍,也再也不会回去了.

​ 若是大一的时候就让我开始面对这些, 我一定会惊慌失措吧, 毕竟, 工作是如此遥远的事. 那时候我刚渡过漫长的暑假.
